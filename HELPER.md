# My Work Timer (V3.7) 使用说明书

**欢迎使用 My Work Timer！** 这是一款专为专注者设计的轻量级桌面工具，旨在帮助您精确记录工作时长、管理番茄钟，并支持灵活的数据同步。

文档撰写: Gemini 3 Pro (AI)（未检查）

------

## 1. 快速入门

### 1.1 启动程序

双击运行 `WorkTimer.exe`。

- **首次运行**：程序会自动在当前目录下生成数据文件 `work_data.json` 和配置文件 `app_pointer.json`。
- **无需安装**：本程序为绿色版，可直接放在 U 盘或网盘文件夹中运行。

### 1.2 主界面概览

程序界面简洁直观，分为四个主要区域：

1. **工作记录区**：显示当前计时大字、工作状态、今日累计时长。
2. **番茄专注区**：设置专注时长、倒计时显示。
3. **底部状态栏**：最小化提示、彻底退出按钮。
4. **顶部菜单栏**：包含设置选项和统计报表入口。

------

## 2. 核心功能指南

### 2.1 记录工作时长

这是最基础的功能，用于统计你“坐在电脑前真正工作”的时间。

1. **开始计时**：点击界面中部的 **[开始工作]** 按钮。
   - 按钮变为“停止工作”。
   - 状态栏变为绿色，显示“工作中 (自 HH:MM)”。
   - 计时器开始跳动。
2. **停止计时**：工作结束后，再次点击 **[停止工作]**。
   - 本次记录会自动保存。
   - **“今日累计”** 时长会自动刷新。
   - *注意：小于 1 分钟的记录会被视为误操作，自动丢弃。*

### 2.2 使用番茄钟 (Pomodoro)

使用番茄工作法来保持专注（默认 25 分钟）。

1. **设置时长**：在“时长(分钟)”输入框中输入数字（默认 25）。
2. **启动**：点击 **[启动]** 按钮。
   - **自动联动**：如果您当前未处于“工作状态”，启动番茄钟会自动帮您按下“开始工作”按钮，确保时长被记录。
3. **完成/取消**：
   - **倒计时结束**：程序会弹出提示窗口，并将窗口置顶，提醒您休息。
   - **中途取消**：点击 [取消] 按钮可重置番茄钟。

## 3. 设置与个性化 (重要)

点击菜单栏 **选项** -> **⚙️ 设置 (路径/习惯)** 打开设置面板。

### 3.1 修改数据存储位置 (云同步)

此功能允许您更改数据保存的位置，例如将其改到 **OneDrive** 或 **Dropbox** 文件夹中，实现多台电脑数据同步。

1. 点击设置窗口中的 **[📂 修改/新建 数据文件路径...]**。
2. **选择路径**：
   - **切换现有文件**：选中一个已存在的 `.json` 数据文件 -> **立即切换**（无弹窗）。
   - **新建/空文件**：输入一个新的文件名 -> 弹出 **“数据迁移向导”**。
3. **数据迁移向导**：
   - **是 (Yes)**：将当前累积的所有历史记录复制到新文件中（推荐）。
   - **否 (No)**：创建一个全新的空数据库，重新开始记录。
   - *注意：路径修改是即时生效的，无需点击保存按钮。*

### 3.2 设置“新的一天” (处理熬夜情况)

如果您习惯工作到深夜（例如凌晨 2 点），不希望 0 点一过数据就归零，可以调整此设置。

1. 在 **“新的一天开始于 (凌晨几点)”** 输入框中填入数字。
   - *示例*：填入 `4`。这意味着 **凌晨 03:59** 的工作记录仍算作 **昨天**，直到 **04:00** 才会开始计算新的一天。
2. **必须点击** 输入框右侧的 **[保存]** 按钮，设置才会生效。

## 4. 窗口管理与后台运行

本程序设计为常驻后台运行，避免误关闭。

- **最小化**：点击窗口右上角的 **[×]** 关闭按钮。

  - 程序**不会退出**，而是隐藏到系统托盘（右下角任务栏）。
  - 托盘图标为一个绿色的 **W**。

- **恢复显示**：左键单击托盘区的绿色 **W** 图标。

- **彻底退出**：

  1. 点击主界面底部的 **[彻底退出]** 按钮。
  2. 或者右键点击托盘图标 -> 选择 **退出程序**。

  - *退出时，如果正在计时，程序会自动保存当前进度。*

## 5. 查看统计报表

点击菜单栏 **统计** -> **打开可视化报表**。 *(需确保已有足够的历史数据)*

- **周统计**：查看本周每天的工作时长柱状图。
- **分布图**：查看您的工作习惯主要集中在几点钟。
- **热力图**：(如已实现) 查看月度投入强度。

## 6. 常见问题 (FAQ)

**Q1: 为什么我工作了很久，"今日累计" 还是 0？**

- 请检查您的 **“新的一天开始于”** 设置。如果您设置了凌晨 4 点开始新一天，而现在是早上 3 点，那么您的工作时长会被计入“昨天”。
- 请确认您是否点击了 **[停止工作]**，只有停止时才会结算并刷新显示（或手动重启程序）。

**Q2: 我把 `work_data.json` 移动了位置，程序打开数据没了？**

- 程序找不到文件时会建立新的空库。请进入 **设置** -> **修改路径**，重新定位到您移动后的那个 `.json` 文件即可恢复数据。

**Q3: 界面下方的按钮显示不全？**

- 这是由于 Windows 屏幕缩放（DPI）导致的。我们在 V3.7 版本中已经增加了窗口高度并允许拉伸。您可以尝试用鼠标拖拽窗口下边缘，将其拉长即可。
